타임리브(thymeleaf)는 자바 라이브러리이며,
웹과 환경 양쪽에서 TEXT,HTML,XML, javascript,css를 생성할 수 있는
템플릿 엔진
스프링 mvc와의 통합 모듈을 제공하며, 애플리케이션안에서 JSP로 만든 기능들을 완전히 대체할 수 있다.

1.jsp와 유사하게${}를 별도의 처리없이 이용할 수 있습니다.
2.Model에 담긴 객체를 화면에서 javascript로 처리하기 편리합니다.
3.연산이나 포맷과 관련된 기능을 추가적인 개발없이 지원합니다.
4.개발도구를 이용할 때 .html파일로 생성하는데 문제가 없고 별도의 확장자를 이용하지 않습니다.
